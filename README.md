## Getting Started

Clone the repository and run 
```bash
./initialsetup.sh
```
Then you can run any of the scripts in the following manner 

```bash 
./scriptname.sh any_directory_name_of_your_choice
```
### Prerequisites
Nodejs (Latest version preffered)
npm
